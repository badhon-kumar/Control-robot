# Control-robot

This is a test project for controlling six MyActuator 48V RMD-X8-120 motors (RMD-X8-P20-120-E/C variant) using CAN

#Basic testing using TUI
use python main.py
this will test the codes if they are working or not

#Interact Using GUI - control angles
use python gui.py
this will show a GUi to interact with and also allows to choose simulation or real motors
